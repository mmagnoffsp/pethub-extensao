# 🐾 Guardião Pet SP

Projeto focado na gestão e auxílio à adoção responsável de animais.

## 🚀 Tecnologias Utilizadas

### Core & Interface
* ![Python](https://img.shields.io/badge/python-3670A0?style=flat&logo=python&logoColor=ffdd54) **Python 3.x** - Linguagem principal.
* ![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?style=flat&logo=Streamlit&logoColor=white) **Streamlit** e **Flet** - Para criação de interfaces dinâmicas e dashboards.
* ![Pandas](https://img.shields.io/badge/pandas-%23150458.svg?style=flat&logo=pandas&logoColor=white) **Pandas** - Manipulação e análise de dados dos pets.

### Backend & Banco de Dados
* ![PostgreSQL](https://img.shields.io/badge/postgresql-%23336791.svg?style=flat&logo=postgresql&logoColor=white) **PostgreSQL** - Banco de dados relacional (Hospedado via **Neon** e **Supabase**).
* ![FastAPI](https://img.shields.io/badge/FastAPI-005571?style=flat&logo=fastapi) **FastAPI** - Framework para construção de APIs de alta performance.

### Integrações & Segurança
* **Google Cloud:** Integração com Google Sheets (Gspread) e serviços de autenticação.
* **Segurança:** Validação de dados com **Pydantic** e gerenciamento de tokens com **PyJWT**.
* **Utilidades:** Geração de **QR Codes** para identificação e automação de processos com **GitPython**.

### Ferramentas de Desenvolvimento
* **IDE:** VS Code
* **Versionamento:** Git & GitHub
* **Ambiente:** Virtualenv (venv)
